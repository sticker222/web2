태그 자체를 선택자로 지정할 수도 있고, 태그 안에 속성을 입력하여 새로운 선택자로 만들 수 있다. - 'class' 'id' 선택자.

  • class 선택자는 class="~" 을 태그의 속성에 적으면서 만들어지고, style 태그에서 선택자로 사용할 때는 ~값 앞에 .을 붙여 적으면 된다.
  • id 선택자는 id="~" 을 태그의 속성에 적으면서 만들어지고, style 태그에서 선택자로 사용할 때는 ~값 앞에 #을 붙여 적으면 된다.


*선택자 우선순위 : id > class > element (=태그)
같은 순위의 선택자끼리는 가장 나중에 오는 것이 적용된다.
