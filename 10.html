디자인을 위해, 어떤 의미도 존재하지 않는 태그를 사용하기도 한다.

  • div 태그 : block element
  • span 태그 : inline element

그리드

세로로 배열 된 두 컨텐츠를 가로로 나란히 놓기 위해, 우선 두개의 컨텐츠를 각각 div로 묶어준다. 그리고 두개의 div 태그를 다른 div 태그 밑에 넣는다. 두개의 div 태그를 포함하는 div 태그는 부모태그가 되는 것이다. 이렇게 부모 태그를 만들어 준 다음에, 혼동을 피하기 위해 부모 div 태그에는 id 값을 설정해준다. 예를 들어 id="grid" 라고 해보겠다.
style 태그에서, #grid {display:grid; grid-template-columns:150px 1fr;}
grid-template-columns 뒤에 입력하는 수치는 각 컨텐츠의 차지 공간을 가리킨다. fr은 공간의 차지 비율을 말한다.
