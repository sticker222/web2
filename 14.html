웹페이지에 중복되는 style 태그 안의 코드들은 별도의 CSS 파일로 저장해서 사용할 수 있다.
style 태그를 제외한 css 코드들만 복사해서 .css 파일로 저장한 후 link 태그를 사용하여 html 파일에 작성하면 된다. link 태그를 사용하므로써, .css 파일에 있는, 별도의 파일을 다운 받아서 코드가 원래 있던 것처럼 동작하도록 한다.


여러 페이지에 걸쳐 중복되는 css 코드를 별도의 파일로 담아 사용하므로써 중복의 제거가 가능하고, 수정도 한번에 할 수 있어 용이해진다.
웹페이지 안에 css를 넣는게 더 효율적일 수 있으나 "캐싱" 때문에, 한번 css파일을 다운받았다면 컴퓨터에 저장해두었다가 다시 요청시 저장된 걸 불러오므로 속도를 높일 수 있다.
